# Rent-A-Ride
Rent a Ride is a web application for vehicle rental system developed as part of Software Engineering course.
---------
**Descripton:** It is a rental website from where customers can register to utilize the services and once registered can login and rent vehicles. The site provides features to search for vehicles based on criteria like vehicle type, location and the date on which the vehicle is to be rented. Customer makes a reservation for a vehicle which will transform into a rental depending on whether the customer has picked up the vehicle or cancelled the reservation. Customer's account will be debited based on the rental status. 

Technologies used: Java, Servlets, HTML5, CSS, Freemarker Template Engine, Javascript
